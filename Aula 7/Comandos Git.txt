Comandos GitHub:

cd (diretório) -> Entrar no diretório
cd .. -> Voltar uma pasta
cd mkdir "Nome da pasta" -> Criar uma pasta

git rm "arquivo" -> Apagar um arquivo do diretório
git add . -> enviar todos arquivos de uma vez para o github
git add "nome do arquivo" -> enviar um unico arquivo para o github
git commit -m "Mensagem" -> Fazer commit
git push -> upar pro github
